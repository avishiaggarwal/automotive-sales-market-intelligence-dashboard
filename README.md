🚗 Automotive Sales & Market Intelligence Dashboard
📌 Project Overview

This project analyzes over 548,000 automotive sales transactions to uncover trends in vehicle pricing, market value, sales performance, and inventory characteristics.

The solution demonstrates an end-to-end data analytics workflow using Python, MySQL, and Power BI, from data cleaning to executive dashboard development.

Objectives
Analyze automotive sales performance
Compare selling prices with Manheim Market Report (MMR)
Identify high-performing manufacturers
Study pricing trends across vehicle age and mileage
Build an interactive business dashboard for decision-making
